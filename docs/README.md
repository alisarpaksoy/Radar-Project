# Documentation

This folder contains the written documentation related to the Radar Project.

## Contents

This directory is intended for files such as:

* article drafts
* final article submission files
* project documentation
* technical notes
* reports
* presentation-related documents

## Purpose

The `docs` folder is used to keep all written materials for the project organized in one place. It supports the development of the technical article, project explanation, and other supporting documents related to the radar-like scanning system.

## Suggested Files

Examples of files that may be stored here:

* `Radar Project Documentation.docx`
* `Radar Project Documentation.pdf`

## Related Project Areas

The documents in this folder are related to:

* hardware description
* MATLAB application explanation
* signal processing implementation
* FFT and filtering analysis
* project results and discussion

## Note

This project focuses on a radar-like object detection system using an ESP32, a laser time-of-flight sensor, an SG90 servo motor, and a MATLAB application for visualization and signal processing.
