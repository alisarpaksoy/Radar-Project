# MATLAB App Screenshots

This folder contains screenshots of the MATLAB App interface used in the Radar Project.

## Purpose

The images in this folder document the graphical interface of the MATLAB application and show how the acquired measurement data is visualized during operation.

These screenshots are useful for:

* project documentation
* technical reports
* article preparation
* presentation materials
* GitHub project description

## Included Views

This folder is intended to contain screenshots of the main MATLAB App modes, such as:

* **initial app window**
* **radar view**
* **distance view**
* **angle view**
* **spectrum plot view**
* **raw and filtered distance plot**

## Role in the Project

The MATLAB App is the main software environment for real-time analysis of the radar-like scanning system. The screenshots in this folder help illustrate:

* the structure of the graphical user interface,
* the start and stop control flow,
* the radar-style spatial map,
* the distance-time graph,
* the filtered signal display,
* the FFT-based spectrum analysis,
* the live servo angle plot.
