# Hardware Component Images

This folder contains photographs of the physical hardware components used in the Radar Project.

## Purpose

The images in this folder document the main embedded and sensing elements of the system. They are useful for:

* project documentation
* technical reports
* article preparation
* presentation materials
* GitHub project explanation

## Included Components

The images currently represent the main hardware used in the project, including:

* **ESP32 development board**
  The main control unit responsible for sensor communication, servo control, and serial data transmission.

* **VL53L1X laser time-of-flight sensor**
  The distance measurement sensor used to detect the range to objects in the scanned environment.

* **SG90 micro servo motor**
  The actuator used to rotate the sensor and provide angular scanning.

* **Integrated prototype setup**
  Combined views showing how the ESP32, servo motor, and distance sensor are assembled on the breadboard during testing.

## Role in the Project

These hardware images support the explanation of the physical structure of the radar-like scanning system. They help show how the sensing, actuation, and control layers are implemented in the prototype.

## Note

This folder is intended for hardware photos only. MATLAB App screenshots and result graphs should be stored in their corresponding folders inside the `images` directory.
